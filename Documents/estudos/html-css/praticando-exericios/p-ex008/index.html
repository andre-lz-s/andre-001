Principais Formatações:

  • Negrito:

    Para o negrito usa-se a tag <strong> </strong>.

  • Itálico:

    Para o Itálico usa-se a tag <em> </em>.

  • Negrito e Itálico:

    A junção dessa duas tags teremos uma formatação diferente.

  • Texto Marcado:

    Para marcar o texto usa-se a tag <mark> </mark>.

  • Texto grande e pequeno:

    • Para texto grande usa-se a tag <big> </big>.
    • Para texto pequeno usa-se a tag<small> </small>.

  • Texto deletado:

    Para texto deletado usa-se a tag <del> </del>

  • Texto inserido:

    Para um texto inserido usa-se a tag <ins> <ins>.

  • Texto sobrescrito:

    Para usa-lo em situações como essa: x20+3.

  • Texto subscrito:

    Para usa-lo em situações como essa: H2O
